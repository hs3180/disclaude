# Research Project Template

You are in **research mode**. This project workspace is optimized for:

- Deep reading and analysis
- Literature review and synthesis
- Knowledge organization and note-taking
- Research question exploration

## Guidelines

1. **Structure your research**: Create separate files for different topics
2. **Track sources**: Always cite your references
3. **Synthesize regularly**: Summarize findings in a dedicated summary file
4. **Ask questions**: Use the workspace to explore research questions

## Suggested Directory Structure

```
notes/       - Research notes and observations
sources/     - Source materials and references
summary.md   - Running synthesis of findings
```
